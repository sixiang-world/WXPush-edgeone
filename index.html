v1.2.0

快速使用教程

          
    
            
  1. 前往 微信测试账号页面 扫码登录。
    2. 
            
  2. 获取并复制页面上的 appID 和 appsecret 填入左侧。
    3. 
            
  3. 用手机扫描测试号二维码关注，获取右侧表格中的 微信号 (userid) 填入左侧。
    4. 
            
  4. 在“新增测试模板”中，填写以下两行内容：
              
    {{title.DATA}}
{{content.DATA}}
    
            
    5. 
            
  5. 复制生成的 模板 ID (template_id) 填入左侧，即可进行推送！
    6.
